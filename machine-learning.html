Overview

This section explores how I use machine learning for materials science applications, including data-driven modeling, property prediction, and accelerating simulations.

Machine learning has revolutionized how we approach problems in computational materials science. From regression models predicting elastic constants, to neural networks learning interatomic forces, the integration of ML with physics-based modeling has unlocked powerful insights and speed-ups.

In my work, I focus on applying ML models to predict materials properties, discover novel compounds, and automate simulation pipelines using data collected from experiments and simulations. Libraries such as Scikit-learn, PyTorch, and TensorFlow play key roles in my research pipeline.

Example projects include training graph neural networks to model atomic interactions, or using unsupervised learning to classify phases of materials from simulation outputs.

Projects

A few notable machine learning projects I've worked on in materials science.

Graph Neural Networks for Interatomic Potentials

Phase Classification

Built regression models for thermal conductivity, bulk modulus, and elastic constants.
